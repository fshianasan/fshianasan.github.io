Firefox に MacType を適用する方法。

2015-02-08 +0900

ずっと前から Firefox派 の私ですが、個人的にずっと font のジャギが気になってました。

で、何とかしようとして模索していましたがやっと MacTray を適用する方法が見つかったので記事にしときます。

前提

  • Firefox 36.0
  • MacType 1.2012.1022.0
  • MacTray 1.0.5.1529

解決方法

まず Firefox を開いてアドレスバーのところに about:config と打ち設定画面を開きます。

検索バーに direct2d と打つと検索結果に gfx.direct2d.disabled という項目がありますので、 その値を true と変えて Firefox を再起動すると MaxType が適用されてるのが確認できます。

これで悩んでいた font のジャギ問題が直ってよかったっ
